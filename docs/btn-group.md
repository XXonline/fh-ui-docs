### BtnGroup

功能按钮组件，封装了常用的按钮操作，目前分为两类，左按钮组和右按钮组。

#### 代码示例
左按钮组
```html
<template>
  <left-btn-group></left-btn-group>
</template>

<script>
  export default {}
</script>
```

右按钮组
```html
<template>
  <right-btn-group></right-btn-group>
</template>

<script>
  export default {}
</script>
```
